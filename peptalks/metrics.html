Part 4: Metrics Definition

After PEP Talk prioritization, Community Organizers and Data Scientists define one or more metrics for each prioritized epic problem, independent of proposed solutions.

Metrics Principles

  • Problem metrics are defined before solution proposal requests.
  • Each metric includes measurable success and failure ranges.
  • Baseline data should be collected where applicable.
  • Optional: define metrics for each proposed epic problem at submission time.

Solution Accountability

If a proposed solution fails to move at least one defined metric within the designated time period, it can no longer be counted as a solution, impacting any formal support for that proposal.
